Create content

We offer several authoring tools that allow you to create learning content. They include tools within our learning platforms (GCcampus and Open Learning) and stand-alone authoring tools, such as H5P and the SNAP framework.

On this page, you will find the criteria for publishing your content and details about the content authoring tools we offer.

Available authoring tools

We offer and support the following content authoring tools that meet our publishing criteria listed in the section below.

    H5P       Moodle      SNAP   

Criteria for publishing your content

We pride ourselves on delivering inclusive experiences to all of our learners. This means that our content:

  • Is written in plain language 
  • Includes text, images and examples that represent Canada's diversity
  • Meets the School's official languages requirements
  • Meets the School's accessibility requirements (WCAG 2.0 AA or higher)
  • Is published in an open format for all Canadians to benefit, when possible

H5P

  • Overview
  • Benefits
  • Example

H5P (HTML 5 package) is a free, open-source HTML5 based content creation tool that allows you to create interactive course content directly in Moodle, Brightspace and other web platforms.

  • Does not require technical programming skills to create content
  • Provides content activity builders that meet WCAG 2.1 AA. (more details here: https://documentation.h5p.com/content/1290410474004879128)
  • Creates learning activities (quizzes, tests, branching scenarios) that can be tracked to a learner's transcript
  • Aallows you to create and publish H5P content directly in Open learning and GCcampus
  • Allows you to publish content on one platform and then embed it on multiple platforms like a YouTube video
  • Builds mobile-responsive content

Moodle

  • Overview
  • Benefits
  • Example

Moodle is a free, open-source PHP based learning management system. It provides the most flexible tool set to support both blended learning and 100% online courses. It easily integrates everything needed for a course using its complete range of built-in features, including external collaborative tools such as forums, wikis, chats and blogs.

Moodle was used to build the Open Learning platform, as well as part of GCcampus. The Open Learning platform uses Moodle version 3.10, while GCcampus uses Moodle version 3.5. 

  • Does not require technical programming skills to create content
  • Allows you to enroll and manage a cohort of learners
  • Offers an accessibility checker directly in the content builder area
  • Provides content activity builders that meets WCAG 2.1 AA on the Open learning platform (more details here: https://docs.moodle.org/dev/Accessibility)
  • Allows you to track learner progress on GCcampus
  • Allows you to create and publish content (quizzes, tests, images, text) directly in the platform
  • Provides a repository for course materials and assignments for virtual courses
  • Publishes entire courses in .mob format that can be uploaded to other Moodle platforms
  • Builds mobile-responsive content

SNAP framework

  • Overview
  • Benefits
  • Example

The SNAP framework is an open-source code library for building innovative e-learning that is accessible, usable, interoperable, mobile-friendly and multilingual. The framework is based on the Web Experience Toolkit (WET) and bootstrap. This collaborative open-source project is led by the School. You can follow the link to see view the SNAP code repository on Github: https://github.com/snap-framework/snap.

Content created in SNAP is published as a SCORM package.

  • Builds a multilingual experience directly in the content that allows your learner to toggle to the equivalent section in French or English at any point
  • Creates content that meets WCAG 2.0 AA
  • Creates content that allows you to track learner progress when published on GCcampus
  • Allows your course developer to create and publish different types of content (quizzes, tests, images, text)
  • Builds mobile-responsive content

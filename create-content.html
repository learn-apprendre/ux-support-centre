Create content

We offer several authoring tools that allow you to create learning content. They range from tools included in our learning platforms (Moodle and Brightspace) to stand alone authoring tool such as H5P, Learn-o-matic and Pressbooks.

On this page, you will find a summary of the criteria for publishing your content and information on the content authoring tools we offer.

Criteria for publishing your content

We pride ourselves on delivering inclusive experiences to all of our learners. This means that our content:

  • Is written in plain language.
  • Includes images and examples that represent Canada's diversity.
  • Is offered in an accessible format that strives to reach WCAG 2.0 AA or higher.
  • Is published in an open format for all Canadians to benefit, when possible.

H5P

What is H5P?

  • A free, open source HTML5 based content creation tool which allows you to create interactive course content directly in Moodle, Brightspace and other web platforms.
  • Provides content activity templates that meet WCAG 2.1 AA
  • It enables learning activities (quizzes, tests, completions) to be tracked online.
  • It is available to build content in Open learning and GCcampus.
  • Allows you to publish content on one platform and then embed it on multiple platforms like a YouTube video.
